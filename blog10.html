Bank Management System – C++ Project (Terminal Based)

This project was created as part of our first semester computer programming subject. It is a terminal-based Bank Management System written in C++. The aim of the project was to simulate basic banking operations like creating accounts, depositing money, withdrawing money, and calculating zakat.

Introduction

Managing customer accounts and transactions manually becomes difficult as the number of customers increases. To solve this issue, we developed a simple and user-friendly Bank Management System using C++. It can perform basic banking tasks through a text-based menu system.

Objectives

  • Allow customers to register and create new bank accounts.
  • Enable secure deposits and withdrawals from accounts.
  • Support for different account types like savings and checking.
  • Calculate zakat automatically on applicable accounts.
  • Store and display account summaries and reports.

System Features

1. Account Creation

Users can create new bank accounts by providing required information like name and initial deposit.

2. Deposit and Withdrawal

The system allows users to deposit or withdraw money by entering the account number and the amount.

3. Zakat Calculation

For savings accounts with a balance above a certain limit, zakat is calculated and displayed.

4. Reports

The system shows detailed account information like account number, holder name, and balance.

Functional Requirements

Customer Management

  • Create and store customer details securely.
  • View and update customer information easily.

Account Management

  • Open and close accounts.
  • Support multiple types of accounts.
  • Perform deposits, withdrawals, and calculate zakat.

Transaction Handling

  • Update balances in real-time after every transaction.
  • Display history of transactions if required.

Technologies Used

  • Language: C++
  • Platform: Windows Terminal (Command Line)
  • IDE: Visual Studio / Code::Blocks

System Design Overview

The project has different modules:

  • Create Account: Accepts user data and creates a new account.
  • Deposit: Adds money to the selected account.
  • Withdraw: Subtracts money from the selected account.
  • Report: Displays account information and balance.
  • Zakat: Calculates zakat based on account type and balance.

Sample Code Snippet


class BankAccount {
  string name;
  int accountNumber;
  double balance;

public:
  void createAccount() {
    cout << "Enter Name: ";
    cin >> name;
    cout << "Enter Account Number: ";
    cin >> accountNumber;
    cout << "Enter Initial Deposit: ";
    cin >> balance;
  }

  void deposit(double amount) {
    balance += amount;
  }

  void withdraw(double amount) {
    if (amount <= balance) balance -= amount;
    else cout << "Insufficient balance!";
  }

  void display() {
    cout << "Account Number: " << accountNumber << endl;
    cout << "Name: " << name << endl;
    cout << "Balance: " << balance << endl;
  }
};
