Understanding Inheritance in Python – Object-Oriented Concept

This blog explains the concept of inheritance in Python using simple examples. Inheritance allows us to create a new class from an existing class. The new class can reuse the methods and properties of the old class. This helps reduce code repetition and keeps our program clean and organized.

Introduction

In object-oriented programming, inheritance means one class (called child class) gets the properties and methods of another class (called parent class). For example, a Student and an Instructor are both types of Person, so we can write common code in the Person class and let the other classes inherit from it.

Why Use Inheritance?

  • To avoid writing the same code again and again.
  • To organize the code better using base (parent) and derived (child) classes.
  • To make programs easier to maintain and update.

Basic Syntax of Inheritance


class Parent:
    # parent class code

class Child(Parent):
    # child class code

Example: Person, Student, Instructor

Let’s create a Person class with common properties. Then we will create two more classes: Student and Instructor that inherit from Person.

Person Class


class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def display_info(self):
        print("Name:", self.name)
        print("Age:", self.age)

Student Class (inherits from Person)


class Student(Person):
    def __init__(self, name, age, student_id):
        super().__init__(name, age)  # call the parent constructor
        self.student_id = student_id

    def display_student(self):
        self.display_info()
        print("Student ID:", self.student_id)

Instructor Class (inherits from Person)


class Instructor(Person):
    def __init__(self, name, age, subject):
        super().__init__(name, age)
        self.subject = subject

    def display_instructor(self):
        self.display_info()
        print("Subject:", self.subject)

Using the Classes


# Creating a student object
s1 = Student("Ali", 20, "ST123")
s1.display_student()

# Creating an instructor object
i1 = Instructor("Dr. Ahmad", 45, "Mathematics")
i1.display_instructor()

Output:


Name: Ali
Age: 20
Student ID: ST123

Name: Dr. Ahmad
Age: 30
Subject: Mathematics

Learning Outcome

In this example, we learned how to use inheritance in Python. The Person class stores common information. The Student and Instructor classes reuse this information and add their own extra data. This makes the code clean, simple, and easy to reuse. We also learned about the super() function which is used to call the parent class constructor.

Conclusion

Inheritance is a powerful feature in Python. It allows us to build complex systems by reusing code. Once you understand how parent and child classes work, you can build large applications with better structure. Practice this by creating more classes like Employee, Manager, Vehicle, etc., and applying inheritance to them.
