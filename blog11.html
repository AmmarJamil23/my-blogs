How to Install Node.js and npm (Latest Version) on Ubuntu

If you are a web developer or computer science student learning React, Express.js, or building modern web apps, you need to install Node.js and npm.

In this blog, I will show you the **recommended method** using a Node Version Manager (n), and also a backup method using NodeSource. I’ll also explain how users in **Pakistan** can improve their download speed using a local mirror.

Method 1: Install Node.js using Node Version Manager (n) – Recommended

This is the best way to install and manage different versions of Node.js. You can switch between Node versions easily using this method.

Step 1: Install Prerequisites

sudo apt update
sudo apt install curl build-essential -y

Step 2: Install Node.js and npm via n

First install n globally:

sudo npm install -g n

Then install the latest stable version of Node.js:

sudo n stable

You can also install the latest LTS (long-term support) version:

sudo n lts

Step 3: Verify Installation

node -v
npm -v

Step 4: Use Different Node Versions (Optional)

If you want to switch between versions:

sudo n 18.19.0   # install and use specific version

Note for Pakistani Users

If you are from Pakistan, sometimes the download from the main Node.js server is slow. You can switch to the **Pakistani UET mirror** to speed up installation.

To do this, instead of downloading directly from deb.nodesource.com, use:

curl -fsSL https://mirror.pknode.uet.edu.pk/setup_20.x | sudo -E bash -

This server is hosted locally in Pakistan and usually gives better speed and less chance of timeout during setup.

Method 2: Install Node.js from NodeSource (Alternate Method)

If you do not want to use the version manager, you can install Node.js directly from NodeSource:

curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

Which Method Should You Use?

  • Use Method 1 (n) if you want to switch between versions and need flexibility.
  • Use Method 2 if you just want one version quickly without setup tools.

Conclusion

Node.js and npm are essential tools for modern web development. Whether you are working with React, Next.js, Vue, or server-side projects like Express — Node.js helps you build, test, and deploy your applications.

If you're in Pakistan and downloads are slow, remember to use a mirror server to save time and avoid frustration.

You are now ready to start your Node.js development journey on Ubuntu!
