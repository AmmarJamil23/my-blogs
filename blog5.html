This project is a visual and interactive simulation of the classic algorithmic challenge known as the Josephus Problem. It was developed as part of our Design and Analysis of Algorithms coursework using web technologies: HTML, CSS, and JavaScript.

Problem Statement

There are n people standing in a circle, numbered from 1 to n. Starting with the first person, every second person is eliminated in clockwise order. This continues until only one person is left. The problem asks: which position should you stand in to be the last person remaining?

Project Objective

The main goal of this project is to create a fun, interactive way to understand how the Josephus problem works. Instead of solving it just mathematically, we simulate the step-by-step elimination process visually.

Users can enter the number of people, and the program will:

  • Display all people in a circular layout.
  • Eliminate every second person in real time.
  • Log each elimination and show the final survivor.

3. JavaScript (Logic)

The logic of the Josephus elimination is written in JavaScript:

  • On clicking start, people are arranged in a circle using trigonometric math.
  • A loop runs to eliminate every second person (k = 2).
  • The process stops when only one person is left ‚Äî the survivor.
  • A ‚Äúpop‚Äù sound is played during each elimination to make the process more interactive.

let index = 0;
let count = 0;

interval = setInterval(() => {
  let skipped = 0;
  while (true) {
    index = index % n;
    if (people[index].alive) {
      skipped++;
      if (skipped === 2) break;
    }
    index++;
  }

  const target = people[index];
  target.alive = false;
  target.element.remove();
  popSound.play();
  logElimination(target.index + 1);

  count++;
  if (count === n - 1) {
    clearInterval(interval);
    const survivor = people.find(p => p.alive);
    highlightSurvivor(survivor);
  }
  index++;
}, 700);
